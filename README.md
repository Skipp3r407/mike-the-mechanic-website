# Mike The Mechanic - Volusia

A premium demo lead-generation website for a local mobile mechanic business serving Volusia County, Florida.

Built with **Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS**, and **lucide-react** icons.

## Features

- Bold automotive theme (black, charcoal, red, white, metallic gray)
- Fully responsive (mobile, tablet, desktop)
- Sticky mobile call button + click-to-call / mailto links
- Smooth scrolling navigation with a sticky, blur-on-scroll navbar
- On-scroll reveal animations (IntersectionObserver, respects reduced motion)
- Lead-generation contact form (demo submission)
- SEO-friendly metadata, Open Graph tags, and `AutoRepair` JSON-LD structured data

## Sections

Hero · Services · Why Choose Mike · About · Reviews · Same-Day Help Banner · Service Area · Contact · Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command         | Description                       |
| --------------- | --------------------------------- |
| `npm run dev`   | Start the development server      |
| `npm run build` | Create a production build         |
| `npm run start` | Run the production build          |
| `npm run lint`  | Run ESLint                        |
